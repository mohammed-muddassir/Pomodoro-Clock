# Pomodoro-Clock

 - Pomodoro Clock built using ReactJS
 
## About :
 - You can set both session and break time.
 - Maximum limit is 60 mins and Minumum is 1 min. for both.
 - Timer notifies a beep sound when the session time ends.
 - And break time starts automatically.
